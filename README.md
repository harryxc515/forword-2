# 🤖 Telegram Forward Bot (Render Ready)

Node.js Telegram bot that sends `/name` messages to Owner and Sudo Users.

## Features
✔ /name command  
✔ Owner message forwarding  
✔ Sudo user message copy  
✔ Help button  
✔ Support channel button  
✔ MongoDB support  
✔ Render PORT server included  

## Commands

/start - Start bot  
/name - Send message to owner & sudo users  
/help - Show help  

## Deploy on Render

1. Upload project to GitHub
2. Create new Web Service on Render
3. Add environment variables from `.env.example`
4. Deploy 🚀
